## 从零开始的Koa实战

### 实战目的

- 使用 Koa2 实现一个博客系统，支持内容发布、图片上传、文件导出/下载等，最终发布。

- 对以往做的实战内容进行查漏补缺，温故知新。

- 如果时间允许，会使用 Egg 进行重构。

### 技术和要点

- 后端使用 Koa2 及其中间件、数据库使用 MongoDB、 前端使用 React+Redux+UI框架待定。

- 要点：RESTful API、 JWT、 自动生成API文档、混合加密传输、访问权限控制、PM2等。
